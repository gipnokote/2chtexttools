С помощью 2ch.hk text tools можно генерировать код для надписей псевдографикой на имиджбордах.

Идея и оригинальная реализация на Java под Android принадлежит одному неизвестному, но безусловно прекрасному Анону. Исходники были предоставлены другому Анону, перепилившему ее на JavaScript. Выпьем за опенсорс, родная!
